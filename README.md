# 旅游分享（travelshare）

## 这是一个前后端分离的项目，由于前后端都是自己开发，所以可能项目比较粗糙，但是也基本将所学到的vue框架方面的知识都串了一遍

## 

## 下载项目后运行以下命令初始化项目
```
npm install
```

### 运行项目
```
npm run serve
```

### 将项目进行打包
```
npm run build
```

### 项目测试
```
npm run test
```

### 代码规范检查
```
npm run lint
```

### 进行项目单元测试
```
npm run test:unit
```
